# AI 3D CAD Model
### "Cursor for CAD"

-Generates 3D CAD Models from user text prompts
-The application uses Google's Gemini API to convert text descriptions into OpenSCAD code, which is then compiled to STL files. You can view, transform, and export your  models through an interactive web interface built with Next.js and Three.js.
-Allows you to export to .stl files


## Tech Stack

- **Backend**: FastAPI (Python)
- **Frontend**: Next.js with Three.js
- **AI**: Google Gemini API
- **3D**: OpenSCAD for model generation
